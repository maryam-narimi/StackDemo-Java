# Stack Implementation (Dynamic Memory)

A simple Java program that demonstrates the use of a **Stack** implemented using **LinkedList**.  
The program allows users to push and pop elements interactively through console input.

## Features
- Stack implementation using LinkedList  
- Dynamic memory allocation  
- User input for stack operations  
- Displays stack after each operation

## Example Output
``Enter number of elements to push: 3
Enter element 1: 10
Enter element 2: 20
Enter element 3: 30
Stack after pushes: [30, 20, 10]
Popped element: 30
Stack after pop: [20, 10]``
